# TSX 파일 분석

특정 TSX 파일을 SrcXray 관점으로 분석합니다.

## 사용법

```
/analyze-tsx [파일 경로 또는 코드]
```

## 분석 항목

1. **Import 분류**: 외부 패키지 vs 내부 파일
2. **컴포넌트 구조**: Props, Hooks, JSX 태그 목록
3. **함수 목록**: 컴포넌트 함수 vs 유틸리티 함수
4. **학습 포인트**: 초보자가 이해해야 할 핵심 패턴

## 출력 형식

- 500줄 이상: 섹션별 요약 우선
- 500줄 미만: 전체 구조 분석
- 핵심 패턴은 코드 예시와 함께 설명

## 관련 파일

- `src/lib/parser.ts` — AST 파싱 로직
- `src/components/` — 분석 결과 표시 컴포넌트
