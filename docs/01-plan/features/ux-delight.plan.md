# Plan: ux-delight — 엔드유저 감동 업그레이드

**작성일**: 2026-02-23
**레벨**: Dynamic
**목표**: 이미 완성된 분석 기능 위에, 유저가 "이거 완전 잘 만들었다" 감탄하게 만드는 UX 감동 포인트 추가

---

## 현재 상태 요약

### 구현 완료된 핵심 기능
| 기능 | 설명 |
|------|------|
| F-07 샘플 버튼 | 빈 화면에서 즉시 체험 |
| F-08 체크박스 진행 | 학습 가이드 체크 + 완료 배너 |
| F-09 카운트업 애니메이션 | 분석 결과 숫자 카운트업 |
| F-10 훅 번역 탭 | 훅 비유 카드 |
| F-11 패턴 동기부여 | TldrCard에 "이걸로 뭘 만들 수 있나" |

| F-13 위험 신호 감지 | 버그 패턴 감지 + 해결책 |
| F-14 필수 개념 카드 | 개념 발견 카드 |
| F-15 useEffect 타임라인 | 실행 시점 시각화 |

### 아직 없는 것 (= 감동 포인트 공백)
- 분석 결과와 실제 코드 줄의 연결이 없음 → "어느 줄?" 모름
- 분석 결과를 저장하거나 공유할 방법 없음
- 이전에 분석한 파일 기억 없음
- "이 파일 다 봤다" 이후 다음 행동 안내 없음

---

## 사용자 페르소나

**왕초보 개발자** (기존 유지)
- 분석 카드를 보고 "어느 줄인지 바로 보고 싶다"
- 분석 결과를 친구에게 보여주고 싶다
- "오늘 이 파일 분석했다"는 기록이 남으면 뿌듯하다
- "이제 뭘 더 배워야 하나?" 방향이 없어서 멈춤

---

## 신규 기능 (WHAT)

### F-17: 🎯 카드 → 코드 줄 하이라이트 연동
**"이 카드가 말하는 게 바로 저 줄이구나!"**

- 위험 신호 / 필수 개념 / 훅 카드 클릭 → 코드 textarea 해당 줄로 스크롤 + 하이라이트
- 하이라이트: 해당 줄 배경색 강조 (1.5초 후 fade out)
- 구현: 분석 결과에 이미 줄 번호가 있음 → textarea scrollTop 계산
- 라인 번호 영역도 동기화

**기대 반응**: "클릭하니까 바로 그 줄로 가네!! 완전 편하다"

---

### F-18: 📋 분석 요약 복사
**"친구한테 보내고 싶다"**

- TldrCard에 "📋 결과 복사" 버튼 1개
- 클릭 시 텍스트로 포맷된 분석 요약 클립보드 복사:
  ```
  🔬 SrcXray 분석 결과 — UserList.tsx
  ───────────────────
  📏 42줄 | 🧩 컴포넌트 2개 | ⚡ 훅 3개 | 📦 Import 5개
  📌 한 줄 요약: 사용자 목록을 가져와서 검색 필터로 보여주는 컴포넌트
  ⚠️ 위험 신호 1개 | 📖 필수 개념 4개 발견
  🔗 srcxray.vercel.app
  ```
- 복사 후 "복사됨 ✓" 피드백 (1.5초)

**기대 반응**: "오 이거 슬랙에 바로 붙여넣을 수 있겠다"

---

### F-19: 🕐 최근 분석 히스토리 (localStorage)
**"저번에 분석한 그 파일 또 보고 싶다"**

- localStorage에 최근 5개 분석 기록 저장 (파일명 + 코드 + 분석 시각)
- 빈 화면에 "최근 분석한 파일" 섹션 표시
- 클릭 시 코드 로드 (분석은 직접 실행)
- 각 항목: 파일명 + 줄 수 + 분석 시각 + 삭제 버튼

**기대 반응**: "어 저번에 한 거 또 볼 수 있네"

---

### F-20: 🎓 "다음 학습 추천" 카드
**"이제 뭘 배워야 하나?" 방향 제시**

- 학습 가이드 100% 완료 시 또는 TldrCard 하단에 표시
- 발견된 개념 기반으로 다음 학습 추천:
  | 발견된 개념 | 추천 다음 학습 |
  |-----------|--------------|
  | useState | useReducer (복잡한 상태), Context API (전역 상태) |
  | useEffect | React Query (서버 상태), SWR |
  | props | TypeScript 제네릭, Zod 검증 |
  | form | React Hook Form, Zod |
  | async/await | TanStack Query, 에러 바운더리 |
- 카드 클릭 시 공식 문서 또는 학습 링크 (새 탭)

**기대 반응**: "오 다음에 이거 배우면 되겠다! 방향이 생겼다"

---

### F-21: ✨ 분석 완료 순간 마이크로 애니메이션
**"결과가 나타나는 순간 감동"**

- 분석 결과 첫 렌더 시 각 섹션 순차적 fade-in (staggered)
  - TldrCard → 0ms
  - Stats → 100ms
  - Tabs → 200ms
- `animate-fadeInUp` 커스텀 keyframe (Tailwind 확장)
- 탭 전환 시도 cross-fade 효과

**기대 반응**: "결과 나오는 거 보는 것만으로도 기분 좋다"

## 기술 제약

- 브라우저 전용 (오프라인 완전 동작)
- localStorage: 코드 저장 시 용량 주의 (5MB 한도, 최대 5개)
- 외부 라이브러리 추가 없음 (Tailwind keyframe 확장만)
- 기존 분석 결과 타입에 줄 번호 이미 포함되어 있으므로 F-17 파싱 추가 불필요
